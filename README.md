## Instalação

_Irei descrever abaixo como você você importar o projeto localmente na sua máquina_

1. Clone o Repositório
   ```sh
   git clone https://github.com/aaglis/clone-tiktok.git
   ```
3. Instale os Pacotes NPM
   ```sh
   npm install
   ```
4. Inicie o Projeto
   ```sh
   npm start
   ```
